# Example data

This folder contains example data for testing and demonstrating the GD-Ls pipeline.

The example data were derived from publicly available spatial transcriptomic data reported in the following study:

Gong, Y., Haeri, M., Zhang, X. *et al.* Stereo-seq of the prefrontal cortex in aging and Alzheimer’s disease. *Nature Communications* 16, 482 (2025). https://doi.org/10.1038/s41467-024-54715-y

For example, spatial transcriptomic datasets such as GSM8330065_B01806B6 and GSM8330069_B01809A3 can be used to generate bin200 expression matrices for GD-Ls analysis. The example files provided here are intended to demonstrate the expected input format and workflow of the GD-Ls pipeline.

## Data source

Original study: Gong, Y., Haeri, M., Zhang, X. *et al.* Stereo-seq of the prefrontal cortex in aging and Alzheimer’s disease. *Nature Communications* 16, 482 (2025).  
DOI: https://doi.org/10.1038/s41467-024-54715-y  
Example GEO sample IDs: GSM8330065_B01806B6 and GSM8330069_B01809A3

## Included example data

The example data in this folder are representative bin200 matrices generated from publicly available spatial transcriptomic datasets, such as GSM8330065_B01806B6 and GSM8330069_B01809A3. These files were not newly generated in the present study and are provided only to facilitate testing and demonstration of the GD-Ls analysis workflow.

## Usage note

These example data are provided for reproducibility, testing, and workflow demonstration purposes only. Users who wish to reuse the original data should refer to the original publication and the corresponding public repository records, and should cite the original dataset and publication appropriately.

Please cite both the original dataset/publication and this GitHub repository when using these example data together with the GD-Ls pipeline.
